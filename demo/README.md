# The AI Engineering Crew — Claude Code Demo

A scrollable single-page HTML presentation covering the full Claude Code setup for this project: 7 specialized agents, 3 automated pre-hooks, persistent agent memory, and a mandatory 7-phase feature delivery pipeline.

## Viewing the presentation

Open `index.html` directly in any browser — no server needed (all assets loaded from CDN).

```bash
open demo/index.html
```

Or serve locally:

```bash
cd demo && python3 -m http.server 8888
# → http://localhost:8888
```

## Navigation

Scroll normally. A progress bar at the top of the page tracks your position. The TOC sidebar (visible on wide screens) highlights the current section. The `↑ Top` button (bottom-right) returns to the hero.
